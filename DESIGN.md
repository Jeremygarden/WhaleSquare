# WhaleSquare Design System

## Color Palette
- Background: `#0f1117`
- Surface: `#1a1d2e`
- Border: `#2a2d3e`
- Text Primary: `#e2e8f0`
- Text Muted: `#94a3b8`
- Accent: `#3b82f6`
- Positive: `#22c55e`
- Negative: `#ef4444`
- Highlight/Gold: `#f59e0b`

## Typography
- UI Font: **Inter** (400–700)
- Numeric/Ticker Font: **Roboto Mono** (400–600)
- Use `font-variant-numeric: tabular-nums` for column alignment

## Spacing Scale
- 4px: tight inline spacing
- 8px: compact spacing
- 12px: card and form gaps
- 16px: standard padding
- 24px: section spacing
- 32px: page gutters

## Table Density Rules
- Headers: 12px vertical padding, muted text
- Rows: 12px vertical padding, 1px bottom rule only
- Hover: subtle accent wash (8% opacity)
- Sticky header for data-heavy tables

## Component Patterns
- **Cards:** dark surface, 1px border, soft shadow, 8px radius
- **Navigation:** sticky, subdued links, accent on hover
- **Filters:** dark input shells, search icon prefix
- **Metrics:** label (uppercase, muted), bold value, small subtext
- **States:** skeleton loading with shimmer, empty state with icon, error card with red border
